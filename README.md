## 📝 요약

채용 공고를 크롤링한 뒤 벡터 DB에 저장, **RAG 기반 검색 시스템**을 구현한 디스코드 봇입니다.

---

## 💡 주요 기능

- Playwright 라이브러리를 이용한 **크롤링 구현**
- PostgreSQL을 이용한 **크롤링 데이터 저장**
- **Local LLM & FAISS DB**를 활용한 RAG 검색 구현
- **Linux 사설 SSH 서버 구축**  
  (크롤링, DB, Discord bot 운영용)
- **Discord bot API**를 사용한 프론트엔드 제공

---

## 💻 Tech Stack

- Python
- FAISS
- Langchain
- Discord API
- Playwright
- SQL
- Linux
